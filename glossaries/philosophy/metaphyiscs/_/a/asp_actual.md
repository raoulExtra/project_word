# Actual

## Definition
Actual refers to that which exists or occurs in fact, as opposed to what is possible, potential, or merely apparent. It denotes the concrete, existing state of affairs in the world.

## Details
- **Purpose**: Distinguish real existence from possibility or appearance
- **Key aspects**:
  - Actual vs possible (existence vs potential)
  - Actualism (only actual entities exist)
  - Groundedness (connected to concrete reality)
- **Types**:
  - Actual world (the one that is realized)
  - Actual infinities (completed infinite totalities)
  - Actuality (state of being fully realized)
- **Role**: Fundamental to modal logic, ontology, and the philosophy of existence

## Context
In metaphysics, the distinction between actual and possible is central to modal logic and the analysis of possibility, necessity, and existence. Actualist positions hold that only actually existing entities are real, while possibilist positions allow for the meaningful discourse about non-actual entities.
