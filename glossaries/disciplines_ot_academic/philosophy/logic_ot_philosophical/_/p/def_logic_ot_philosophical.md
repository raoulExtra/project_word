# Philosophical Logic

## Definition
Philosophical logic is the study of logical concepts and methods from a philosophical perspective, emphasizing the analysis of natural language arguments, conceptual analysis, and the application of logical principles to philosophical problems such as meaning, reference, and knowledge.

## Details
- **Purpose**: Bridge formal logic with natural language reasoning and philosophical inquiry
- **Key aspects**:
  - Informal logic (argumentation theory, fallacies)
  - Formal logic (propositional, predicate, modal systems)
  - Philosophical logic (intensional contexts, presupposition)
- **Types**:
  - Deductive logic (valid argument forms)
  - Inductive logic (probabilistic reasoning)
  - Abductive logic (inference to best explanation)
  - Epistemic logic (knowledge and belief)
  - Deontic logic (obligation, permission)
- **Role**: Provides tools for rigorous philosophical analysis while addressing limitations of purely formal approaches

## Context
Philosophical logic differs from mathematical logic in emphasizing interpretation, context, and the analysis of actual human reasoning rather than purely formal systems. It investigates how logical concepts apply to natural language, ethics, metaphysics, epistemology, and the philosophy of mind.
