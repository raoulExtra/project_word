# Functional

## Definition
Functional refers to the purpose, role, or intended operation of a system or entity, focusing on what it is designed to do, the capabilities it provides, and how it achieves its objectives through inputs, processes, and outputs.

## Details
- **Purpose**: Define and analyze the capabilities, roles, and intended outcomes of systems and entities
- **Key aspects**:
  - Function (what the system does)
  - Capacity (limits of capability)
  - Purpose (intended goal or outcome)
  - Input-output relationships
- **Types**:
  - Primary function (main intended purpose)
  - Secondary function (additional capabilities)
  - Emergent function (unexpected capabilities)
  - Dysfunctional (failure to perform intended function)
- **Role**: Provides the framework for understanding why systems exist and how they achieve their intended outcomes, complementing structure (how it's built) and behavior (how it acts).

## Context
In core concepts, functional analysis is essential for ontology engineering, systems modeling, and understanding how entities are designed to operate. While structure describes composition and behavior describes actions, functional describes purpose and capability.
