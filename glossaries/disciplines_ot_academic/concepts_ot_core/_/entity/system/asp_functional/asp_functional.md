# Functional

## Definition
Functional aspect defines the purpose, role, or operation of a system component within the broader system context - what the system does to achieve its goals.

## Details
- **Purpose**: Understand how system components serve system objectives and interact to produce outcomes
- **Key aspects**:
  - System functions (input-processing-output relationships)
  - Functional architecture (organization of capabilities)
  - Performance requirements (criteria for successful function)
  - Functional dependencies (interconnections between components)
- **Types**:
  - Declared functions (intended purposes)
  - Actual functions (observed behaviors)
  - Emergent functions (unexpected capabilities)
  - Dysfunctional states (deviations from intended function)
- **Role**: Enables systematic analysis of system behavior, identification of design alternatives, and evaluation of system effectiveness

## Context
Functional analysis shifts focus from "what is" to "what does," complementing structural analysis with purpose-driven understanding.
