# Behavioural

## Definition
Behavioural aspect defines how a system responds to inputs, changes over time, and exhibits dynamic responses under various conditions and stimuli.

## Details
- **Purpose**: Capture the dynamic, time-dependent responses and reactions of a system
- **Key aspects**:
  - System behaviors (observable responses and actions)
  - State transitions (changes in system condition over time)
  - Input-output mappings (causal relationships)
  - Temporal patterns (sequences, cycles, trends)
- **Types**:
  - Reactive behaviors (responses to external stimuli)
  - Autonomous behaviors (self-directed actions)
  - Emergent behaviors (complex patterns from simple interactions)
  - Degraded behaviors (failure modes, anomalies)
- **Role**: Enables understanding of system dynamics, predictability, and adaptive capacity

## Context
Behavioural analysis bridges the gap between static structure and purposeful function, revealing how systems actually operate in practice.
