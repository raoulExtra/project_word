# Function

## Definition
The purpose, role, or operational task that a system or its components are designed to perform.

## Details
- **Purpose**: Define what a system is meant to accomplish
- **Key aspects**: Input-output mapping, mission, objectives
- **Types**: Primary function, secondary function, emergent function
- **Role**: Guides design and evaluation of system effectiveness
