# System

## Definition
A set of interrelated components working together to achieve a common goal. A collection of interacting components performing functions.

## Details
- **Purpose**: Provide structured approach to complex problems
- **Key aspects**: Components, interactions, boundaries, inputs/outputs
- **Types**: Natural systems, artificial systems, social systems, technical systems
- **Role**: Foundation for systems thinking and analysis

## Context
Systems are fundamental to understanding how complex entities function and interact.
