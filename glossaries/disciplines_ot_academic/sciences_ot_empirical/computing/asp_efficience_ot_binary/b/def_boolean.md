# Boolean

## Definition
A boolean is a data type that can have one of two values, typically representing true/false, yes/no, or 1/0 states in computing systems.

## Details
- **Values**: true/false, 1/0, yes/no
- **Purpose**: Fundamental logical unit for decision-making and conditional operations
- **Relation to bit**: A bit's two states represent boolean values (0=false, 1=true)
- **Operations**: AND, OR, NOT, XOR

## Context
In the computational sequence, boolean is the semantic layer built on top of bits, enabling logical operations and decision-making in computing systems.
