# Bit

## Definition
A bit (binary digit) is the smallest unit of data in computing, representing a single binary value: 0 or 1.

## Details
- **Symbol**: b
- **Values**: 0 (off/false) or 1 (on/true)
- **Purpose**: Fundamental building block for all digital data and computation
- **Relationships**: 
  - 8 bits = 1 byte
  - Bits are combined to represent numbers, text, images, and all digital information

## Context
In the computing domain sequence, the bit is the foundational concept that enables all higher-level computational abstractions.
