# Constraint

## Definition
A constraint is a limiting condition or restriction that defines boundaries or rules governing the behavior, structure, or possible states of a model or system, reducing degrees of freedom and shaping the solution space.

## Details
- **Purpose**: Limit model behavior to reflect real-world limitations
- **Key aspects**:
  - Hard constraints (must be satisfied)
  - Soft constraints (preferred but not required)
  - Boundary constraints (limiting values)
- **Types**:
  - Equality constraints (x = y)
  - Inequality constraints (x < y)
  - Logical constraints (if-then conditions)

## Context
Constraints are fundamental in model specification, ensuring that models reflect realistic limitations and produce meaningful, feasible solutions.
