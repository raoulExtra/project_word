# Applicability

## Definition
Applicability refers to the range of contexts, conditions, or problem domains for which a model can be legitimately applied, indicating the scope of its useful operation.

## Details
- **Purpose**: Define where and when a model can be used effectively
- **Key aspects**:
  - Domain limits (problem space boundaries)
  - Condition requirements (necessary preconditions)
  - Scale constraints (size, time, complexity limits)
- **Types**:
  - Universal applicability (broadly useful)
  - Domain-specific applicability (limited scope)
  - Context-dependent applicability (conditional use)
- **Role**: Determines practical utility and prevents misuse

## Context
Applicability is crucial for responsible model use. A model with narrow applicability may still be highly valuable within its domain, while a broadly applicable model offers greater utility.

**Related terms**: model, boundary, domain
