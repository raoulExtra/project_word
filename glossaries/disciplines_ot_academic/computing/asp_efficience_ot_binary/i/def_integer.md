# Integer

## Definition
An integer is a whole number (positive, negative, or zero) represented in binary form using the word size of the system.

## Details
- **Representation**: Binary encoding (two's complement for signed)
- **Range**: Depends on word size (-32,768 to 32,767 for 16-bit, etc.)
- **Purpose**: First numeric abstraction built on words
- **Operations**: Arithmetic operations (addition, subtraction, etc.)

## Context
In the computational sequence, integers are words interpreted as numbers, enabling arithmetic and numeric processing.
