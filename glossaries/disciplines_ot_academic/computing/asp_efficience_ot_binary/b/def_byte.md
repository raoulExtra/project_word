# Byte

## Definition
A byte is a group of 8 bits, typically used as the smallest addressable unit of memory in most computer systems.

## Details
- **Size**: 8 bits (historically varied: 6, 7, 9, 10, 11, 12, 16, 18, 20, 24, 32, 36, 40, 48, 50, 60, 64, 80 bits)
- **Purpose**: First meaningful grouping of bits for storage
- **Usage**: Represents characters (ASCII), small integers, and basic data units
- **Relation**: 8 bits = 1 byte

## Context
In the computational sequence, bytes enable storage of characters and small integers, bridging the gap between individual bits and larger data structures.
