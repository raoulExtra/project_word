# Sciences for Systems Glossary

Concepts related to systems thinking and organizational structures.

### o/ - Organizational
- [Organizational](o/asp_organizational.md) - Structure and procedures of organizations

### s/ - Systems
- [System](s/def_system.md) - Set of interrelated components working together
