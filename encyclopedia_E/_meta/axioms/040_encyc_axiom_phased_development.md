## Axiom

Each phase of focus development must be describable and implementable using concepts from the same phase or from phases below (more foundational).

## Rationale

Knowledge cannot effectively build upon concepts that are defined later or at the same level. A coherent progression requires that each phase draws only from established foundations.

This axiom ensures:
- No circular dependencies between phases
- Progressive accumulation of understanding
- Foundational stability for higher-level concepts
- Traceable reasoning chains

## Implementation

- Phase N must use concepts from phases 0 to N
- Phase N must not introduce concepts from phases N+1 or beyond
- Documentation must explicitly reference foundational concepts
