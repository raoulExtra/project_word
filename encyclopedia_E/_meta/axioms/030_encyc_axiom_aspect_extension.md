## Axiom

Each concept can be extended through explicit, formally defined aspects that tailor its focus to the current thinking process.

## Rationale

Concepts alone are insufficient for precise reasoning. Aspects provide the dimensions through which concepts can be interpreted and applied.

This axiom ensures:
- Concepts can be enriched without creating new articles
- Aspect-based interpretation remains consistent
- Dynamic focus adaptation is possible
- Cross-domain application is enabled

## Implementation

- Aspects must be formally defined before application
- Each aspect must relate to the core concept
- Aspect activation must be explicit in context
