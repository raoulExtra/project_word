## Axiom

Each distinct concept within a domain under focus has exactly one canonical article.

## Rationale

Knowledge fragmentation occurs when the same concept is represented by multiple articles, creating redundancy and ambiguity. A unified encyclopedia must establish clear identity for each concept.

This axiom ensures:
- No duplicate articles for the same concept within a domain
- A single source of truth for each distinct idea
- Clear resolution paths when concepts overlap domains
- Consistent linking and referencing across the encyclopedia

## Implementation

- Before creating a new article, search for existing canonical representations
- When concepts span domains, use cross-domain linking (see `020_encyc_axiom_cross_domain_linking.md`)
- Canonical articles must explicitly declare their domain scope
