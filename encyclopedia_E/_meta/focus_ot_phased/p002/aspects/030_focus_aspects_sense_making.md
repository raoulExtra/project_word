# Sense-Making Aspects

Aspects derived from phase 1 ground elements to enhance understanding and interpretation:

## Compositional Aspects
- **Conceptual** (from Concept) - how ideas combine and relate
- **Systemic** (from System) - how parts form wholes and patterns
- **Aspectual** (from Aspect) - how perspectives shape understanding
- **Knowledge-based** (from Knowledge) - how understanding is structured
- **Informational** (from Information) - how data becomes meaningful
- **Semantic** (from Meaning) - how symbols connect to significance
- **Framing** (from Framework) - how context shapes interpretation
- **Modeling** (from Model) - how abstractions represent reality
- **Relational** (from Relationship) - how connections create meaning
- **Structural** (from Structure) - how organization enables comprehension

## Application
Each aspect can be applied to any concept to understand it through that lens:
- A *Concept System* viewed through **Conceptual** aspect reveals how ideas interconnect
- A *Knowledge Structure* viewed through **Knowledge-based** aspect shows understanding patterns
- A *Semantic Network* viewed through **Relational** aspect exposes connection dynamics
