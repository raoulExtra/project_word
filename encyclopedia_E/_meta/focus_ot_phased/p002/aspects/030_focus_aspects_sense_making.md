# Sense-Making Aspects

Aspects derived from phase 1 ground elements to enhance understanding and interpretation:

## Compositional Aspects

- **asp_conceptual** (from Concept) - how ideas combine and relate
- **asp_systemic** (from System) - how parts form wholes and patterns
- **asp_aspectual** (from Aspect) - how perspectives shape understanding
- **asp_knowledge_based** (from Knowledge) - how understanding is structured
- **asp_informational** (from Information) - how data becomes meaningful
- **asp_semantic** (from Meaning) - how symbols connect to significance
- **asp_framing** (from Framework) - how context shapes interpretation
- **asp_modeling** (from Model) - how abstractions represent reality
- **asp_relational** (from Relationship) - how connections create meaning
- **asp_structural** (from Structure) - how organization enables comprehension

## Application

Each aspect can be applied to any concept to understand it through that lens:
- A *Concept System* viewed through **asp_conceptual** aspect reveals how ideas interconnect
- A *Knowledge Structure* viewed through **asp_knowledge_based** aspect shows understanding patterns
- A *Semantic Network* viewed through **asp_relational** aspect exposes connection dynamics
