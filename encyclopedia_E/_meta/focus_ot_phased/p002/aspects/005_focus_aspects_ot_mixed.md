## Mixing of Aspects from previous Phase

### Theoretical + Practical + Applied
- **asp_theoretical** + **asp_practical** → **asp_applied** (theory in practice)
- **asp_theoretical** + **asp_applied** → **asp_practical** (theory refined through application)

### Core + Specific
- **asp_core** + **asp_specific** → **asp_particular** (general to specific instances)
- **asp_principal** + **asp_specific** → **asp_operational** (core principles in action)

### Abstract + Structural + Logical
- **asp_abstract** + **asp_structural** → **asp_formal** (abstract structures)
- **asp_abstract** + **asp_logical** → **asp_conceptual** (logical abstractions)
- **asp_structural** + **asp_logical** → **asp_systemic** (logically coherent structures)
