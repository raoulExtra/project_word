# Philosophy Foundations Glossary

Concepts and aspects related to foundational principles and facts.

## Concepts

- [Fact](fact.md) - Verified true statement about the world
- [Foundational](asp_foundational.md) - Basic principles and assumptions

## Overview

Foundational concepts address what underlies philosophical inquiry: the basic elements that must be accepted or understood before more advanced analysis can proceed.
