# Philosophy Core Glossary

Core concepts and aspects in philosophical inquiry.

## Concepts

- [Reasoning](r/reasoning.md) - Cognitive process of generating conclusions
- [Knowledge](k/knowledge.md) - Justified true belief about the world
- [Truth](t/truth.md) - Accurate representation of reality
- [Deductive](d/deductive.md) - Reasoning from general to specific with certainty
- [Abductive](a/asp_abductive.md) - Reasoning to the best explanation
- [Inductive](i/asp_inductive.md) - Reasoning from specific to general with probability
- [Critical](c/asp_critical.md) - Evaluating assumptions and validity
- [Foundational](f/asp_foundational.md) - Basic principles and assumptions
- [Fact](f/fact.md) - Verified true statement about the world
- [Informal](i/asp_informal.md) - Everyday reasoning outside formal systems
- [Philosophical](p/asp_philosophical.md) - Pertaining to fundamental questions
