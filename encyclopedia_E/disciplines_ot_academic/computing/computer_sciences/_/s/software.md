# Software

## Definition
A collection of computer programs, data, and instructions that tell a computer what to do and how to do it.

## Details
- **Purpose**: Provide instructions for computer hardware to perform useful tasks
- **Key aspects**: Programs, data, documentation
- **Types**: System software, application software, programming software
- **Role**: Bridge between human needs and computer hardware capabilities

## Context
Software is one of the fundamental concepts in computer science, enabling the abstraction of hardware complexity into usable tools.
