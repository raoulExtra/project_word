# Hardware

## Definition
The physical devices and components of a computer system that execute software instructions and store data.

## Details
- **Purpose**: Provide the physical foundation for computing
- **Key aspects**: Processing units, memory, storage, input/output devices
- **Types**: Central Processing Unit (CPU), memory (RAM), storage (HDD/SSD), peripherals
- **Role**: Execute software instructions and provide physical interfaces for computing

## Context
Hardware and software are complementary: hardware provides the physical substrate while software provides the logical instructions for computation.
