# Word

## Definition
A word is the CPU's natural processing unit, typically 16, 32, or 64 bits, representing the native data size the processor handles most efficiently.

## Details
- **Sizes**: 16-bit, 32-bit, 64-bit (common modern sizes)
- **Purpose**: CPU's native data processing unit
- **Relation**: Built from bytes (2, 4, or 8 bytes depending on architecture)
- **Usage**: Stores integers, memory addresses, and machine instructions

## Context
In the computational sequence, words define the architecture's "native size" and determine the range of integers and addresses the system can handle.
