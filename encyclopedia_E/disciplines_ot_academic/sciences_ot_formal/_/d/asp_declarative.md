# Declarative

## Definition
Declarative refers to a programming paradigm or logical form that expresses the desired result without specifying the control flow, focusing on "what" rather than "how."

## Details
- **Purpose**: Express logic without procedural details
- **Characteristics**:
  - Statements about relationships
  - No side effects or control flow
  - Focus on truth conditions
- **Examples**: SQL queries, Prolog rules, logical formulas
- **Relation to imperative**: Complementary paradigm emphasizing specification over execution

## Context
In formal sciences and computer science, declarative approaches enable reasoning about systems at a high level of abstraction, separating problem definition from solution strategy.
