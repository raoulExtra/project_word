# Axiom

## Definition
An axiom is a statement or proposition that is regarded as being self-evidently true or assumed to be true within a formal system, serving as a starting point for deducing other truths.

## Details
- **Purpose**: Foundational truth for logical deduction
- **Characteristics**:
  - Not proved within the system
  - Self-evident or accepted on faith
  - Starting point for theorems
- **Types**:
  - Logical axioms (principles of reasoning)
  - Non-logical axioms (domain-specific assumptions)
- **Role**: Defines the scope and structure of a formal system

## Context
In formal sciences like mathematics and logic, axioms provide the irreducible foundation upon which entire theoretical frameworks are built.
