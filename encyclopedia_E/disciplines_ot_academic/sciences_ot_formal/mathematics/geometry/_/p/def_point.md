# Point

## Definition
A point is a fundamental geometric object that represents a location in space, having no dimensions (length, width, or height).

## Details
- **Characteristics**: Zero-dimensional, represents position
- **Properties**:
  - Has no length, area, volume, or other spatial extent
  - Defined by coordinates in coordinate systems
  - Can be named by a single letter or coordinate pair/triple
- **Role**: Building block of geometry (lines, angles, shapes)

## Context
In geometry, points are the most basic element used to define lines, planes, and higher-dimensional objects. They form the foundation for all geometric constructions and proofs.
