# Bootstrapping

## Definition
Bootstrapping is a self-starting process that uses existing resources to create new resources, often involving incremental improvement through successive approximations.

## Details
- **Purpose**: Initialize systems with no prior knowledge or minimal starting point
- **Key concepts**:
  - Self-application (using tool to build better version of itself)
  - Recursive improvement
  - Minimal viable starting point
- **Examples**:
  - Compiler bootstrapping (compiler written in language it compiles)
  - Statistical learning (model trains on its own predictions)
  - Economic development (initial capital generates more capital)

## Context
In computational and technical contexts, bootstrapping enables systems to overcome the "chicken and egg" problem of needing existing capabilities to create better capabilities.
