# Probability

## Definition
Probability is the measure of the likelihood that an event will occur, quantified as a number between 0 and 1.

## Details
- **Measure**: Ranging from 0 (impossible) to 1 (certain)
- **Key concepts**: Random variables, distributions, expectation, variance
- **Foundations**: Measure theory, stochastic processes
- **Applications**: Statistics, gaming, risk assessment, machine learning

## Context
Probability theory provides the mathematical framework for uncertainty quantification and statistical inference.
