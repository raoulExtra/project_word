# Mathematical Logic

## Definition
Mathematical logic applies formal logical systems to mathematics, studying the nature of mathematical proof and reasoning.

## Details
- **Focus**: Propositional logic, predicate logic, proof theory
- **Key concepts**: Formal systems, model theory, computability
- **Foundations**: Set theory, type theory, category theory
- **Applications**: Foundations of mathematics, computer science

## Context
Mathematical logic provides rigorous frameworks for expressing and proving mathematical theorems, and underlies automated reasoning systems.
