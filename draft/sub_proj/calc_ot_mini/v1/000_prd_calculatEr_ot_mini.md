# Calculator Mini

## Overview
A minimal calculator implementation using a simplified accumulator-based architecture.

## Goals
- Implement basic arithmetic operations (ADD, SUB, MUL, DIV)
- Support simple memory operations
- Provide both CLI and programmatic interfaces

## Requirements
- 16-bit word size
- Accumulator-based architecture
- Operation set defined in `requi/010_groups_for_operations.md`

## Constraints
- No floating-point support
- Limited memory (100 addresses)
- Integer arithmetic only
