# Subcommand

## Definition
A nested command within a main command (e.g., `git commit`, `docker build`). Enables complex CLI tools with multiple operations.

## Details
- **Purpose**: Organize complex CLI tools into multiple operations
- **Key aspects**: Nested structure, separate argument sets
- **Role**: Enable multi-operation CLI tools

## Context
Subcommands allow a single CLI tool to support multiple operations with different argument requirements.
