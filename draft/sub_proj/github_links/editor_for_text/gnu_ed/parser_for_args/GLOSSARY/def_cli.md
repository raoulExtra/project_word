# Command-Line Interface (CLI)

## Definition
The text-based interface for interacting with programs. Users enter commands and arguments as text strings.

## Details
- **Purpose**: Provide text-based program interaction
- **Key aspects**: Text commands, argument passing, output display
- **Role**: Bridge between user intent and program execution

## Context
CLIs are the standard interface for many development and system tools, requiring robust argument parsing for usability.
