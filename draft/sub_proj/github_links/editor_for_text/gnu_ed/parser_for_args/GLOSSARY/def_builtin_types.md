# Built-in Types

## Definition
The supported primitive data types for automatic conversion: `int`, `float`, `bool`, `str`, and `list`. Custom type converters are not supported.

## Details
- **Purpose**: Enable type-safe argument handling without manual conversion
- **Types**: int, float, bool, str, list
- **Role**: Automatic type conversion for common CLI use cases

## Context
Built-in types provide the foundation for argument parsing without requiring developers to write custom conversion logic.
