# Help Generation

## Definition
Automatic creation of usage documentation and argument descriptions when users invoke the CLI with `--help` or `-h`.

## Details
- **Purpose**: Provide built-in documentation for CLI tools
- **Key aspects**: Usage strings, argument descriptions, examples
- **Role**: Self-documenting CLI tools

## Context
Help generation eliminates the need for separate documentation by providing context-sensitive help from argument metadata.
