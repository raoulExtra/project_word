# Argument

## Definition
A piece of data passed to a command-line program when it is invoked. Arguments can be positional (order matters) or optional (flags with names like `--verbose`).

## Details
- **Purpose**: Pass input data to CLI programs
- **Types**: Positional arguments, optional arguments (flags)
- **Role**: Primary input mechanism for command-line tools

## Context
Arguments are the fundamental unit of CLI interaction, forming the interface between user intent and program execution.
