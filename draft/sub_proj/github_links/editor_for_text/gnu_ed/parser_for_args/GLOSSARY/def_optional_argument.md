# Optional Argument

## Definition
An argument that is not required and typically has a flag name (e.g., `--output`, `-v`). Also known as flags or options.

## Details
- **Purpose**: Provide optional configuration without breaking existing workflows
- **Key aspects**: Flag names, default values, optional presence
- **Role**: Flexible configuration mechanism

## Context
Optional arguments allow users to customize program behavior without mandatory input.
