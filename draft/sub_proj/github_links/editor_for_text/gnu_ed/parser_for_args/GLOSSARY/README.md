# Python Argument Parser Glossary

This glossary defines key terms for the Python argument parser project.

## Terms

- [Argument](def_argument.md) - Data passed to CLI programs
- [ArgumentParser](def_argument_parser.md) - Main class for parsing arguments
- [Built-in Types](def_builtin_types.md) - Supported primitive types (int, float, bool, str, list)
- [CLI](def_cli.md) - Command-Line Interface
- [Namespace](def_namespace.md) - Container for parsed argument values
- [Optional Argument](def_optional_argument.md) - Flags/options that are not required
- [Parse Tree](def_parse_tree.md) - Internal representation of parsed arguments
- [Positional Argument](def_positional_argument.md) - Arguments identified by position
- [Subcommand](def_subcommand.md) - Nested commands within a main command
- [Type Conversion](def_type_conversion.md) - Converting strings to Python types
- [Help Generation](def_help_generation.md) - Automatic help message creation
