# Positional Argument

## Definition
An argument that is identified by its position in the argument list rather than by a flag name.

## Details
- **Purpose**: Provide required input without flag prefixes
- **Key aspects**: Order-dependent, mandatory (unless default/nargs)
- **Role**: Simple input mechanism for required values

## Context
Positional arguments are the simplest form of CLI input, requiring users to provide values in a specific order.
