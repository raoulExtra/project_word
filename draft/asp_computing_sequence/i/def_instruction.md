# Instruction

## Definition
An instruction is a binary pattern that tells the CPU what operation to perform, encoded as opcodes and operands.

## Details
- **Format**: Binary pattern specifying operation and data
- **Components**: Opcode (operation) + operands (data addresses/registers)
- **Purpose**: Directs CPU execution flow
- **Dependencies**: Requires registers, memory, and words to function

## Context
In the computational sequence, instructions are the highest-level concept, telling the CPU how to manipulate all lower-level components (registers, memory, words).
