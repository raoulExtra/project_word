# Memory

## Definition
Memory is a large array of bytes addressed by numbers (integers), providing storage for data and instructions.

## Details
- **Structure**: Linear array indexed by addresses (integers)
- **Content**: Stores bytes, words, and program instructions
- **Purpose**: Persistent storage for the computing system
- **Dependencies**: Requires addresses (integers) and bytes

## Context
In the computational sequence, memory is the storage layer that holds all data and instructions, addressed by integer indices.
