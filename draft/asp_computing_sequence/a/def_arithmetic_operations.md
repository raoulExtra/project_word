# Arithmetic Operations

## Definition
Arithmetic operations are basic mathematical operations (addition, subtraction, multiplication, division) performed on integers.

## Details
- **Operations**: ADD, SUB, MUL, DIV
- **Input**: Integers (words interpreted as numbers)
- **Output**: Results stored in registers or memory
- **Dependencies**: Requires integers, words, and registers

## Context
In the computational sequence, arithmetic operations are the first functional operations built on top of integers, enabling numeric computation.
