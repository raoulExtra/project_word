# Design Decision: First Domain - Computation

## Decision
We start our first domain as **computation** with the first sequence word being **bit**.

## Rationale
- Computation provides a concrete, foundational domain that maps well to word-level concepts
- Starting with "bit" (the smallest unit of data) establishes a clear entry point for the sequence
- This creates a layered model: bit → instruction → program → etc.
- Enables building a coherent conceptual progression from hardware abstraction to software

## Implications
- First word introduced in the system is "bit"
- Establishes the foundational layer for computational thinking
- Supports future expansion into other domains (linguistics, mathematics, etc.)
